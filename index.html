HTML and CSS Table References

HTML Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<tfoot> Table Foot The set of rows defining the footer in a table.

HTML Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.

Semantic HTML Elements

Tag Name Description
<head> Head A container for metadata and links to scripts and stylesheets.
<header> Header A container for navigational links or introductory content such as <h1> to <h6>.
<nav> Nav Used to define block of navigational links, such as menus and tables of content. Note: Nav can be used inside <header>, but can also be used on its own.
<main> Main Used to encapsulate dominant content within a webpage.
<footer> Footer Typically contains information for: contacting, copyright, Terms of Use, Sitemapping, Reference to Top of Page Links.
<article> Article Holds content that makes sense on its own, such as: articles, blogs, comments, and magazines.
<section> Section Defines elements in a doc, such as: chapters, headings, and any other area of doc with the same theme.
<aside> Aside Used to mark info that can enhance another element but isn't required in order to understand the main content. Common uses: Bibliographies, Endnotes, Pullquotes, etc.
<figure> Figure An element to encapsulate media such as an image, illustration, diagram, code, snippet, etc.
<figcaption> Figure Caption An element to add a caption to a figure.
<audio> Audio To embed audio content. Uses src to link to an audio source. Comes with following attributes: controls, src, and autoplay controls.
<video> Video To add video and can come with the following attributes: controls, autoplay, and loop.

CSS Specificities

Specificities Name Description
!important Important Overrides any style specified below. Should almost never be used. Can be applied to specific declarations instead of full rules.
* Universal Selects all element of any type. Overrides all other specificities below. Can help reset default browser styling or selecting all children of a parent element.
type Type Least specific attribute to style specific elements through CSS.
class Class Mid-specific attribute to style specific elements through CSS. To select a class in CSS, begin with "."
ID ID The most specific attribute to style specific elements through CSS. Use sparingly, as it overrides types and classes. To select an ID in CSS, begin with "#".

Common CSS Properties

Property Name Description
color Color This property styles an element's foreground color.
background-color Background Color This property styles an element's background color.
font-family Font Family This property changes the font of the element. You can enter multiple fonts to ensure all browsers recognize a font for readability.
font-size Font Size Resizes text to a specified pixel (px) size.
font-weight Font Weight This property boldens text.
text-align Text Align This property aligns text. Can use left, center, right, and justify.
opacity Opacity This property measures how transparent you want an element to be. 1 is visible, or opaque, while 0 is invisible.
width Width This property alters the width of a content area in pixels (px).
height Height This property alters the height of a content area in pixels (px).
padding Padding This property alters the amount of space in between the content area and its border. Has padding-top, padding-right, etc.
border Border This property alters the thickness and style surrounding the content area and padding.
margin Margin Alters the amount of space between a border and the outside edge of the element. Has margin-top, margin-right, etc.
border-radius Border Radius This property modifies the corners of an element's border box to be rounded. Use pixels (px).
min-width Minimum Width This property modifies the minimum width an element can have in pixels (px).
max-width Maximum Width This property modifies the maximum width an element can have in pixels (px).
min-height Minimum Height This property modifies the minimum height an element can have in pixels (px).
max-height Maximum Height This property modifies the maxmimum height an element can have in pixels (px).
overflow Overflow This property controls what happens to content that spills, or overflows, outside its content box. Commonly used values are: hidden, scroll, visible.
visibility Visibility This property controls whether certain elements will be hidden from view on the webpage. Can be set to following values: hidden, visible, and collapse.
box-sizing Box Sizing/Content-Box This property controls the type of box model that the browser should use when interpreting the webpage. Can be set to: content-box, border-box (recommended).
position Position This property changes the position of an element. It can use the following values: static, relative, absolute, fixed, and sticky.
z-index Z-Index This property controls how far back or how far forward an element should appear on the webpage when elements overlap. Thought of as "depth" of elements. Accepts integer values. 1 moves the element "higher."
display Display Every display element in HTML dictates that it can share horizontal space with other elements. Its values are: inline, block, inline-block, and none.
float Float This property is commonly used to move elements as far left or right as possible. Must have a width specified. Its values are: left and right.
clear Clear This property specifies how elements should behave when they bump into each other on a page. Has values: left, right, both, none.
